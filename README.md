# Graphics Package Algorithms

A desktop application that demonstrates fundamental computer graphics algorithms.  
The application allows users to draw different geometric shapes and apply basic 2D transformations in a simple graphical interface.

## 📌 Overview
This project was developed to visualize how classic computer graphics algorithms work.  
Users can interact with the application to draw shapes such as lines, circles, and ellipses, and perform 2D transformations.

## ✨ Features
- Draw Line using graphics algorithms
- Draw Circle
- Draw Ellipse
- Apply 2D Transformations
- Simple and interactive graphical interface

## 🖥️ Application Interface
The interface provides a side menu that allows the user to select different graphics algorithms and display the result on the drawing canvas.

## 🛠️ Technologies Used
- C#
- Windows Forms
- .NET Framework

## 🎯 Purpose of the Project
This project was created as part of studying **Computer Graphics** to understand how graphical shapes are generated using mathematical algorithms instead of built-in drawing tools.
